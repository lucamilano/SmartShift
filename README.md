## 🚀 Panoramica / Overview

🇮🇹 **SmartShift** è un portale web moderno progettato per semplificare la gestione delle presenze in ufficio. Permette ai dipendenti di segnalare i propri giorni lavorativi, di Remote Working (Smartworking), ferie o malattie. Gli amministratori possono avere una panoramica completa del team, gestire l'anagrafica ed esportare i report mensili in pochi clic.

🇬🇧 **SmartShift** is a modern web portal designed to streamline office attendance management. It allows employees to report their working days, Remote Working, holidays, or sick leaves. Administrators get a comprehensive overview of the team, can manage user profiles, and export monthly reports in just a few clicks.

## 🛠️ Tecnologie Utilizzate / Tech Stack

Questo progetto è costruito con l'ecosistema React più moderno: / _This project is built with the most modern React ecosystem:_

- **[Next.js](https://nextjs.org/) (App Router)** - Framework React Full-Stack.
- **[Supabase](https://supabase.com/)** - Database PostgreSQL, Autenticazione (Magic Links) e Row Level Security.
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first styling (con varianti di sistema e custom `dark:` classes).
- **[Shadcn UI](https://ui.shadcn.com/)** - Componenti interfaccia utente di altissima qualità e accessibili.
- **[Lucide React](https://lucide.dev/)** - Iconografia moderna.
- **[date-fns](https://date-fns.org/)** - Manipolazione avanzata delle date (calcolo delle festività e weekend).
- **[ExcelJS](https://github.com/exceljs/exceljs)** - Generazione server-side di file Excel con formattazione e colori personalizzati.

## ⚙️ Installazione / Getting Started

<details>
<summary>🇮🇹 <strong>Istruzioni in Italiano</strong></summary>

1.  **Clona il repository:**
    ```bash
    git clone https://github.com/tuo-username/SmartShift.git
    cd SmartShift
    ```
2.  **Installa le dipendenze:**
    ```bash
    npm install
    ```
3.  **Configura le Variabili d'Ambiente:**
    - Copia il file `.env.example` o creane uno denominato `.env.local`
    - Inserisci nel file l'URL del tuo progetto Supabase e la tua chiave Anonima:
    ```bash
    NEXT_PUBLIC_SUPABASE_URL=test_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=test_key
    ```
4.  **Avvia il server di sviluppo:**
    ```bash
    npm run dev
    ```
    Apri [http://localhost:3000](http://localhost:3000) col tuo browser per vedere l'applicazione.

</details>

<details>
<summary>🇬🇧 <strong>Instructions in English</strong></summary>
